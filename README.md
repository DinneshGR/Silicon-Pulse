# 📊 Silicon Pulse BI Project

## 🚀 Overview

**Silicon Pulse** is a 5-page interactive Power BI dashboard that analyzes semiconductor stock performance and market capitalization trends from **2020 to 2025**.

The project focuses on five leading semiconductor companies:

* NVIDIA
* AMD
* Intel
* Broadcom
* Qualcomm

Built using **6,000+ daily trading records**, this dashboard delivers actionable insights into market behavior, growth trends, and company performance.

---

## 🎯 Objectives

* Analyze stock price movements across major semiconductor companies
* Track market capitalization growth over time
* Identify industry trends and performance drivers
* Enable interactive, user-driven financial exploration

---

## 📌 Key Features

### 📊 Interactive Dashboard (5 Pages)

* Stock price trend analysis
* Market capitalization comparison
* Company-wise performance breakdown
* Time-series insights with slicers and filters
* Cross-company comparison visuals

---

### 🧹 Data Preparation & Modeling

* Cleaned raw stock data using **Power Query**
* Handled:

  * Inconsistent date formats
  * Currency formatting issues
* Built a **multi-table relational data model**
* Established relationships for seamless cross-dataset analysis

---

### 📐 DAX Measures Implemented

* 📈 30-Day Moving Average
* 🔄 Daily Return (%)
* 📊 Year-over-Year Market Cap Growth

---

## 💡 Key Insights

* NVIDIA experienced **900%+ market cap growth**, driven by the **AI boom in 2023**
* Clear differentiation in growth patterns among semiconductor leaders
* Market trends highlight increasing dominance of AI-focused chip manufacturers

---

## 🛠️ Tech Stack

* **Power BI** – Data visualization & dashboard design
* **Power Query** – Data cleaning & transformation
* **DAX (Data Analysis Expressions)** – Advanced calculations

---

## 📂 Dataset Details

* **Type:** Historical stock data
* **Time Range:** 2020–2025
* **Records:** 6,000+
* **Structure:** Multi-table relational model
